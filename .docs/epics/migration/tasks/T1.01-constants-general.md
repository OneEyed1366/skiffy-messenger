## Description

Migrate general constants from Mattermost webapp that are needed for UI components and client-side logic. Focus on constants that are platform-agnostic and useful for React Native.

**Included:**

- User statuses
- Item status (for UI feedback)
- KeyCodes (keyboard handling)
- Preferences categories and values
- File types
- Notification levels
- Channel types
- Modal identifiers
- Validation limits

**Excluded:**

- Server-only constants (job types, stat types)
- Web-specific constants (CSS classes, HTML IDs)
- Plugin-related constants
- Admin console paths
- External URLs and links
